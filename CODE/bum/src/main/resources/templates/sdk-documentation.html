Java SDK

Installation

  1. Download BumApiClient.java
  2. Add to your project: src/main/java/com/example/
  3. Add dependency: org.json:json:20231013 to pom.xml

Usage

import com.example.BumApiClient;

// Initialize client
BumApiClient client = new BumApiClient("your-api-key", "https://yoursite.com");

// Validate API key
boolean isValid = client.validateApiKey();

// Validate license
BumApiClient.LicenseResponse response = client.validateLicense("LICENSE-KEY");
if (response.valid) {
    System.out.println("License is valid!");
    System.out.println("Product: " + response.productName);
    System.out.println("Expires: " + response.expireDate);
} else {
    System.out.println("License invalid: " + response.message);
}

Hardware-Locked License

// For hardware-locked licenses
String hardwareId = getSystemHardwareId();
BumApiClient.LicenseResponse response = client.validateLicense("LICENSE-KEY", hardwareId);

if (response.valid) {
    System.out.println("Hardware match!");
}

Python SDK

Installation

  1. Download bum_sdk.py
  2. Install required dependency: pip install requests
  3. Import in your script: from bum_sdk import BumApiClient

Usage

from bum_sdk import BumApiClient

# Initialize client
client = BumApiClient('your-api-key', 'https://yoursite.com')

# Validate API key
is_valid = client.validate_api_key()

# Validate license
response = client.validate_license('LICENSE-KEY')
if response['valid']:
    print('License is valid!')
    print('Product: ' + response['productName'])
    print('Expires: ' + response['expireDate'])
else:
    print('License invalid: ' + response['message'])

Hardware-Locked License

import uuid

# For hardware-locked licenses
hardware_id = str(uuid.getnode())
response = client.validate_license('LICENSE-KEY', hardware_id)

if response['valid']:
    print('Hardware match!')

API Reference

License Validation Endpoint

POST /api/v1/licenses/validate

Request Headers
Authorization: Bearer {your-api-key}
Content-Type: application/json
Request Body
{
    "licenseKey": "LICENSE-KEY",
    "hardwareId": "optional-hardware-id"
}
Response (Success)
{
    "valid": true,
    "status": "active",
    "message": "License is valid",
    "productName": "Product Name",
    "expireDate": "2025-12-31 23:59:59",
    "hardwareId": "hardware-id",
    "userId": "123"
}
Response (Error)
{
    "valid": false,
    "status": "expired",
    "message": "License has expired",
    "productName": "Product Name",
    "expireDate": null,
    "hardwareId": null,
    "userId": "123"
}

Response Status Codes

Status Description
active License is active and valid
expired License has passed expiration date
banned License has been revoked
unused License not yet activated
not_found License key does not exist
hardware_mismatch Hardware ID does not match
unauthorized License does not belong to your product

Error Codes

Error Message Solution
invalid Invalid or inactive API key Check your API key in dashboard
not_found License key not found Verify the license key spelling
expired License has expired Renew license or contact support
banned License has been banned Contact support for more info
hardware_mismatch Hardware ID does not match Use correct hardware ID or request license transfer
rate_limited Rate limit exceeded Cache validation results, wait before retrying

Best Practices

Security

  • Store API key in environment variables, never hardcode
  • Use HTTPS for all API calls
  • Implement request timeout (5-10 seconds)
  • Never expose API key in client-side code
  • Rotate API keys regularly

Performance

  • Cache validation results for 24 hours
  • Implement offline fallback
  • Use async validation where possible
  • Monitor API usage in dashboard
  • Set appropriate rate limits

Development

  • Handle all error cases gracefully
  • Test with test licenses first
  • Implement proper logging
  • Use connection timeouts
  • Validate input before sending
